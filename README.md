Repo for the [accepted project](https://summerofcode.withgoogle.com/programs/2022/projects/YberDUGn) proposal to [CRIU](https://criu.org) in [Google Summer of Code 2022](https://summerofcode.withgoogle.com). Apart from the final submission, it also contains the first draft, in order to allow people to see just how a proposal changes and grows from the initial version to the end result.

You can find the PR of this project [here](https://github.com/checkpoint-restore/go-criu/pull/66).

GitHub does not do a good job of rendering `proposal.md`, so I would suggest viewing the PDF file instead.
All the PDFs were generated using [mdpdf](https://github.com/bluehatbrit/mdpdf) with the following command:

`mdpdf proposal.md --style proposal.css --gh-style`
